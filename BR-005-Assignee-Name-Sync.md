# BR-005 - Assignee Name Sync Across Separate Sessions

This bug report documents a cross-session data consistency issue observed during exploratory testing of a SaaS workflow application.

The issue was initially reproducible when a team member changed their profile name in a separate session, while the Project Owner continued to see the outdated assignee name in the Task List.

The report includes structured reproduction steps, environment details, expected vs actual behavior, and evidence prepared for support investigation.
